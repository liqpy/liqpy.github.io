# 个人博客静态主页

一个美观、响应式的个人博客静态主页，采用纯HTML、CSS和JavaScript实现，无需后端或数据库。

## 🌟 功能特性

### 1. 📱 响应式设计
- 完美适配桌面、平板和手机设备
- 移动端友好的导航菜单
- 自适应布局和字体大小

### 2. 🎨 现代化UI设计
- 卡片式文章布局
- 柔和的阴影和圆角
- 平滑的过渡动画
- 一致的配色方案（蓝色为主色调）

### 3. 🏗️ 完整页面结构
- **头部导航栏**：固定位置，包含响应式菜单
- **博客文章区**：显示最新文章，包含标题、摘要、元数据和阅读更多链接
- **侧边栏**：包含博主信息、文章分类和热门文章
- **页脚**：包含快速链接和联系信息

### 4. ⚡ 交互功能
- 移动端菜单切换
- 文章卡片悬停效果
- 社交链接动画效果

## 📁 文件结构

**注意**：本项目使用单文件架构，所有代码（HTML、CSS、JavaScript）都包含在`index.html`文件中。

## 🛠️ 技术栈

| 技术 | 用途 | 版本 |
|------|------|------|
| HTML5 | 页面结构 | 语义化标签 |
| CSS3 | 样式设计 | Flexbox/Grid布局 |
| JavaScript | 交互功能 | ES6+ |
| Font Awesome | 图标库 | 6.4.0 |
| Unsplash | 高质量图片 | CDN |

## 🚀 快速开始

### 方法一：直接使用
1. 复制`index.html`文件内容
2. 保存为`.html`文件
3. 在浏览器中打开即可

### 方法二：在线体验
1. 下载本项目
2. 双击`index.html`文件
3. 自动在默认浏览器中打开
